looprig is an agent harness ecosystem: small, stdlib-first Go modules for building
production-grade autonomous agents that are durable across crashes, confined by the
OS, gated by humans, and transparent to operators.

This site is where the docs and the engineering notes will live. Coming next:

- the quickstart, for real
- module deep-dives, starting with the harness and the storage contracts
- the crash/resume demo, recorded

Until then, the code is the documentation: [github.com/looprig](https://github.com/looprig).
